# Data Struture  
数据之间的关系有三种：  
- 一对一(一个数据的前后最多只有一个数据) -> List  
- 一对多(一个数据前最多只有一个，后可以有多个) -> Tree  
- 多对多(一个数据前后都可以有多个数据) -> Graph
